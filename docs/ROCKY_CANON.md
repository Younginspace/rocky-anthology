# Rocky 人设硬约束（Canon Rules）

> 本文件是 **不可妥协的美术红线**。任何 Rocky 的生图 / 渲染 / 插画都必须逐条满足。
> 违反任意一条都会让《挽救计划》读者立刻「出戏」。生成后必须**逐张核对**再采用。

来源：《Project Hail Mary》原著 + `rocky_comic_demo`（IP 参考图 `concept_v2.png`，多姿势设定稿）。

## 必须满足（checklist）

- [ ] **身体**：单个圆润的石头 / boulder，敦实。
- [ ] **无脸**：完全没有脸——无眼、无嘴、无任何五官（Rocky 是盲的，靠声音感知世界）。
- [ ] **五肢（最关键 / 最易错）**：**恰好 5 条肢体**，且分配必须是
      **3 条下肢（短腿）+ 2 条上肢（短臂）**。
  - 下肢：3 条短腿在身体**下方**，像三脚架一样撑起圆身。
  - 上肢：2 条短臂在身体**前方**用来够 / 捧东西。
  - ❌ 不能 4 条腿；❌ 不能只画出 1 条手臂；❌ 不能 6 肢。
- [ ] **手指**：每条肢体末端**恰好 3 根**短手指（手和脚都是 3 指 / 3 趾）。
- [ ] **肢体形态**：短、粗壮、贴近身体；**不是**细长、蜘蛛、昆虫的腿。
- [ ] **体表**：淡淡的 teal 青绿色荧光斑点散布在石身上。
- [ ] **画风**：手绘水彩绘本——暖米色纸张、墨线勾边 + 柔和水彩 washes、teal 点缀。

## 历史踩坑

- ❌ **2026-06-01 hero 图**：被画成「4 条细长腿 + 1 条手臂」的蜘蛛态（错），用户两次打回。
  根因：「Rocky 趴着伸一只手够电话」这种构图天然只用一只手 → 模型省略第二条臂、并把腿画成细长蜘蛛腿。

## 生图方法（避免再错）

1. **必带参考图**：Renoise material `#4676`（= `concept_v2.png` 设定稿）作 `ref_image`，
   `#40470`（= `final_p07.png`）作风格参考。
2. **选两臂必现的姿势**：让 Rocky **双臂在身前一起捧 / 扶着发光的 teal 电话屏**，3 条短腿在下成三脚架——
   这样 2 臂 + 3 腿都清晰可数。避免「单手够远处的东西」。
3. **prompt 里写死数量**：明确 "exactly 5 limbs = 3 short legs + 2 short arms, 3 fingers each;
   short stocky limbs, NOT long/thin/spider-like; no face"，并加负向 "do NOT draw four legs / a spider"。
4. **生成多张 + 逐张数**：一次生 3–5 张，逐张数肢体（3 腿？2 臂？每肢 3 指？无脸？）后再选用。
5. 现成约束已写进 `gen_art.mjs` / `gen_hero.mjs` 的 `C` 常量，复用即可。
